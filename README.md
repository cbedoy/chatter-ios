# CHATTER
Basic Objetive - C Implementation [PubNub](https://www.pubnub.com/)

## How to use

```bash
git clone git@github.com:cbedoy/chatter-ios.git chatter
cd chatter
pod install
```

Then just open workspace project and having fun.
